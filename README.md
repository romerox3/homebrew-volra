# Homebrew Tap for Volra

[Volra](https://github.com/romerox3/volra) — Own your agent infrastructure.

## Install

```bash
brew install romerox3/volra/volra
```

Or add the tap first:

```bash
brew tap romerox3/volra
brew install volra
```

## Verify

```bash
volra --version
```

## Update

```bash
brew upgrade volra
```
